## 💡 为什么需要 Fade Reader？

传统的英文阅读工具通常采用“长按查词”的逻辑。每遇到一个生词，你必须：
1. **中断**阅读流。
2. **点击**单词并等待翻译。
3. **记忆**后尝试回到上下文。

这种频繁的打断是“阅读兴趣”的头号杀手。 **Fade Reader 彻底反转了这一流程。**

## 🚀 核心黑科技：“单词自动隐身” (The Fade Mechanic)

Fade Reader 提倡一种 **“反向查词”** 的体验，让阅读从“受挫”变成“闯关”：

1. **默认全显**：打开一本原版书，所有单词下方默认直接显示中文翻译，阅读像中英对照版一样顺滑。
2. **点击“消除”**：当你发现某个单词已经认识了，轻轻一点，它的翻译就会 **瞬间消失（Fade）** 。
3. **留白美学**：随着阅读推进，翻译会被你一个个“消灭”，页面上留下的英文越来越多，最终只剩下纯净的英文内容和零星的生词。
4. **随时召回**：忘记了？右键点击，翻译立刻重新出现，学习过程完全可逆。

> **它把“查生词”的痛苦，变成了“把单词变消失”的成就感。**

## 🌟 主要功能

- 📦 **多格式支持**：完美支持 `EPUB` 和 `Markdown` 原版电子书。
- 📖 **智能分页排版**：自动识别章节，根据窗口大小自适应分页，告别卡顿。
- 🛠️ **自定义词典**：支持上传专业词典（医学、金融、科技等），让翻译更懂你的领域。
- 🤖 **AI 整句翻译**：遇到长难句？右键一键调用 Google / Gemini / OpenAI (自定义 API)，配合 TTS 真人发音，打通听力与理解。
- 📊 **阅读数据量化**：自动统计阅读时长、掌握词汇量、高频生词表，见证你的进步。
- 🔒 **隐私至上**：完全本地运行，不上传任何阅读数据，零联网也能安心使用。
